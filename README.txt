Pengingat Tugas

Isi folder ini:
- pengingat_tugas.html  -> file utama (buka di browser)
- manifest.json         -> (opsional) manifest untuk PWA
- service-worker.js     -> (opsional) service worker dasar
- icon-192.png          -> ikon placeholder
- README.txt            -> file ini

Cara pakai cepat:
1. Ekstrak folder hasil unduhan (jika zip).
2. Buka pengingat_tugas.html di browser seluler/desktop.
3. Tekan tombol 'Aktifkan Audio & Notifikasi' supaya browser memberi izin suara/notifikasi.
4. Tambah pengingat. Untuk PWA: install melalui opsi browser (Add to home screen) jika tersedia.

Catatan:
- Alarm bekerja saat halaman terbuka. Untuk PWA native behavior, perlu pengembangan lebih lanjut.
